Binary Search: Complete Guide with Examples

Date: November 5, 2025 | Difficulty: Medium | Time: 10 min read

Binary Search is one of the most fundamental algorithms in competitive programming. It allows us to search for an element in a sorted array in O(log n) time complexity.

Problem Statement

Problem: Given a sorted array of n integers and a target value, find the index of the target value. If not found, return -1.

Input: arr = [1, 3, 5, 7, 9, 11], target = 7
Output: 3

Approach

The idea is to repeatedly divide the search space in half. We maintain two pointers, left and right, and compare the middle element with the target:

  • If middle element equals target, we found it
  • If middle element is less than target, search in right half
  • If middle element is greater than target, search in left half

Implementation

C++ Code

int binarySearch(vector<int>& arr, int target) {
    int left = 0, right = arr.size() - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;
        
        if (arr[mid] == target) {
            return mid;
        }
        else if (arr[mid] < target) {
            left = mid + 1;
        }
        else {
            right = mid - 1;
        }
    }
    
    return -1;
}

Python Code

def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = left + (right - left) // 2
        
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1

Complexity Analysis

Time Complexity: O(log n) - We divide the search space by 2 in each iteration
Space Complexity: O(1) - Only using constant extra space

Common Mistakes

  1. Integer overflow: Use mid = left + (right - left) / 2 instead of (left + right) / 2
  2. Infinite loop: Make sure to use left <= right not left < right
  3. Off by one errors: Carefully handle the boundary conditions
